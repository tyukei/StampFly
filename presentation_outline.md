### 3. システムアーキテクチャ（5枚）
**スライド6: 全体システム構成**
```
[脳波] → [PiEEG] → [Raspberry Pi] → [UDP] → [ESP32-S3] → [ドローン]
                         ↓
                   [Webダッシュボード]
```

**スライド7: ハードウェア構成**
- **PiEEG**: 16チャンネル脳波センサー
- **Raspberry Pi**: データ処理・通信ハブ
- **ESP32-S3**: ドローン制御マイコン
- **ドローン**: カスタムクアッドコプター

**スライド8: ソフトウェアアーキテクチャ**
- **脳波解析層**: θ/α/β/γ波分析
- **通信層**: UDP/WiFiプロトコル
- **制御層**: PWM/ESC制御
- **可視化層**: WebUI/Chart.js

**スライド9: データフロー**
1. 脳波信号取得（250Hz sampling）
2. 周波数解析（FFT）
3. 特徴抽出（パワー比率）
4. UDP送信（500ms間隔）
5. モーター制御（PWM出力）

**スライド10: 通信プロトコル**
- **UDP通信**: 低遅延（<10ms）
- **データ形式**: JSON/数値
- **エラーハンドリング**: タイムアウト機能
- **セキュリティ**: ローカルネットワーク

### 4. 脳波解析技術（4枚）
**スライド11: 脳波の基礎**
- **θ波（4-8Hz)**: 深いリラックス・瞑想
- **α波（8-12Hz)**: リラックス・集中準備
- **β波（13-30Hz)**: 通常覚醒・思考
- **γ波（30-100Hz)**: 高度集中・認知

**スライド12: 信号処理パイプライン**
```
生信号 → フィルタリング → FFT → パワー計算 → 比率分析 → 制御信号
```

**スライド13: 制御マッピング**
- **θ優勢**: 左回転（リラックス時）
- **α優勢**: 上昇（集中準備時）
- **β優勢**: 直進（通常思考時）
- **γ優勢**: 右回転（高集中時）

**スライド14: ノイズ対策**
- **ハードウェアフィルタ**: 0.5-100Hz帯域
- **ソフトウェアフィルタ**: ノッチフィルタ（50/60Hz）
- **アーチファクト除去**: 瞬き・筋電図対策
- **平滑化**: 移動平均・メディアンフィルタ

### 5. ドローン制御技術（4枚）
**スライド15: ハードウェア制御**
- **ESP32-S3**: 240MHz デュアルコア
- **PWM制御**: 50Hz、1000-2500μs
- **ESC**: 電子スピードコントローラ
- **モーター配置**: クアッドコプター構成

**スライド16: 推力制御システム**
- **通常モード**: 5-100%推力
- **強力モード**: 100-200%推力
- **安全機能**: タイムアウト（3秒）
- **緊急停止**: 最小推力復帰

**スライド17: 飛行制御ロジック**
```cpp
if (eegValue < 1.0f) {
    // 左回転: FL/RL=200%, FR/RR=150%
} else if (eegValue < 3.0f) {
    // 上昇: 全モーター200%
} else {
    // 右回転: FL/RL=150%, FR/RR=200%
}
```

**スライド18: 安全機能**
- **タイムアウト監視**: 3秒で自動停止
- **推力制限**: 最大200%まで
- **緊急停止**: 即座に最小推力
- **電源管理**: バッテリー監視
